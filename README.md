# Zoster
Deeplinking Scheme URL generator, tester for Android. In development.

# Usage
```bash
git clone https://github.com/Urucas/zoster.git && cd zoster
npm start
```
**or**
```bash
npm install -g zoster
zoster
```

# Requirements
* Appium

**Android**
* ADB

**iOS**
* ideviceinstaller
