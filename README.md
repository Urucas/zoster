# Zoster
Url deeplinking automate testing for Android

# Usage
```bash
git clone https://github.com/Urucas/zoster.git && cd zoster
npm install

npm start
// or run electron less, directly in your browser
npm run server
```
**or**
```bash
npm install -g zoster
zoster
// run electron less
zoster --neutron
```

# Requirements
* Appium

**Android**
* ADB
